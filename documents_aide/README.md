# Dossier Rapports — BookCycle Tunisia

Ce dossier contient les rapports finaux du projet intégré **BookCycle Tunisia**, rédigés dans le cadre de la Licence 2 Big Data et Intelligence Artificielle à l'ESEN — Université de la Manouba (2025/2026).

---

## Contenu du Dossier

| Fichier | Module | Description |
|---|---|---|
| `01_RAPPORT_PROJET_INTEGRE.md` | Tous modules | Rapport principal complet : contexte, analyse, SGBD, Web 2, RPA, tests, conclusion |
| `02_RAPPORT_AGL_RPA.md` | AGL · RPA | Analyse des besoins, user stories, backlog, architecture, processus, BPR, scénarios d'automatisation |
| `03_RAPPORT_SGBD.md` | SGBD | Schéma Oracle, contraintes, séquences, triggers, index, vue, SQL, objets PL/SQL |
| `04_RAPPORT_WEB2.md` | Prog. Web 2 | Architecture MVC PHP, connexion Oracle PDO, contrôleurs, modèles, vues, sécurité |
| `05_FICHE_PROJET.md` | Synthèse | Fiche de présentation synthétique — utile pour la soutenance ou une remise rapide |

## Sources et Cohérence

Ces rapports ont été rédigés en cohérence avec :

- La structure réelle du code source dans `app/`
- Le schéma Oracle défini dans `database/02_schema.sql`
- Les objets PL/SQL dans `database/05_plsql_objects.sql`
- Les requêtes SQL dans `database/04_queries.sql`
- Les pages web dans `app/Views/pages/`
- Les contrôleurs dans `app/Controllers/`
- Les modèles dans `app/Models/`
